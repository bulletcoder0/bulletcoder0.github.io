Deze video legt uit hoe een computer werkt.

input

De invoer van een computer kan van alles zijn.
Het kan bijvoorbeeld het intikken van toetsen op je toetsenbord zijn.
Of het kan een geluid zijn dat een microfoon oppikt.
Basicly alle informatie die een computer van buiten krijgt is input


Opslag en verwerking

Nadat een computer input krijgt wordt deze input verplaatst naar de opslag.
Als de informatie in de opslag zit wordt deze informatie verwerkt.
De informatie vordt steeds opnieuw verwerkt en opgeslagen totdat de informatie helemaal verwerkt is.


Output

Als de informatie klaar is met verwerken dan komt deze informatie weer naar buiten als output.
Output kan bijvoorbeeld het verschijnen van een letter op je scherm zijn.
Of het kan ook een geluid zijn dat uit de computer komt.
