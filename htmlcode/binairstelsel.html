Het binaire stelsel.

Als een computer input krijgt dan is deze input in bits.
Een bit is een 1 of een 0 Dit op zichzelf is niks maar als je er meerdere hebt dan kan je er wat mee doen.
Als je acht bits op een rij zet krijg je een byte en met een byte kan je het binaire stelsel gebruiken.
Bij het binaire stelsel met een byte kan je 255 mogelijkheden hebben om een getal te krijgen.
De manier waarop je met het binaire stelsel telt is alvolgd:
De eerste bit heeft een waarde van 1.
De tweede bit heeft een waarde van 2.
De derde bit heeft een waarde van 4.
De vierde bit heeft een waarde van 8 en dit gaat alsmaar door.
Je kan vervolgens waarden optellen met deze bits want als een bit op 0 staat dan tel je hem niet mee .
En als een bit op 1 staat dan tel je hem wel mee.
Bijvoorbeelt de code 1010 is 10 en de code 0110 is 6.
Op deze manier kan je met maar een paar bits heel veel mogelijke nummers uitbeelden.


Maar hoe beeld je dan getallen uit met bits en bytes?

Als je met het binaire stelsel getallen wilt uitbeelden is daar een ander binair alfabet voor.
Bij dit binaire stelsel gebruik je altijd een hele byte
Bijvoorbeelt: 01100001 is a en 01100010 is b.
Als je een getal wilt uitbeelden in het binaire stelsel dan doe je 011 en dan het getal 1-26 in binaire taal.
en dan heb je een getal. En als je een hooftletter wilt uitbeelden dan doe je 010 en dan het getal 1-26 in binaire taal.
Al deze sijfers en nummers staan allemaal in het ASCII tabel.
