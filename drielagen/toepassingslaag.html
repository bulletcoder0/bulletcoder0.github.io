Api's


Een Api is een interface die het mogeljk maakt om twee aplicaties met elkaar te laten comuniceren.
Een api wordt gebruikt als je bijvoorbeeld als je online iets wilt bestellen op een website zoals bol.com.
Je voert bijvoorbeeld in dat je een grijze jas wilt bestellen.
Om alle goede resultaten voor jouw zoekopdracht te vinden moet je de imformatie hebben over alle producten die met je desriptie overeen komen.
De computer comuniceert dan met de api van de website om toegang te krijgen tot deze imformatie.
Dus in conclusie: een api is een interface die informatie vanaf jouw computer naar de systemen van een website stuurt en daarna weer een antwordt treug stuurt.
