Ram


Ram staat voor random access memory en is een onmisbaar onderdeel in een computer.
Het ram geheugen zorgt er bijvoorbeeld voor dat de cursor beweegt of dat je kan typen en ook dat je dingen kan opslaan.
Tijdens het gebruik van een computer speelt het ram geheugen als breug tussen onderdelin in de computer.
Het ramgeheugen wordt dan bijvoorbeeld gebruikt om tijdelijk informatie op te slaan die nodig is om een ander onderdeel in de computer te gebruiken.

Rom


Rom staat voor read onley memory.
Rom wordt gebruikt om informatie op te slaan zelfs als de computer niet aanstaat.
Er zijn verschillende soorten rom zoals:

Prom


Prom is een Rom die leeg is en waar dus nog informatie in geprogrameert kan worden

Eprom


Een Eprom is een speciale soort rom chip de weer gereset kan worden door er ultraviolet licht op te schijnen.
Eprom is handig om te gebruiken om te kijken of code werkt want een normale Rom of Prom kan niet gereset woorden maar een Eprom wel.

Eeprom


Eeprom heeft als voordeel op zijn voorganger ( de Eprom ) dat de chip gereset kan woorden met electriesche spanningen,
wat het mogelijk maakt om de inhoud te wijzigen terwijl de chip nog in de computer zit.
