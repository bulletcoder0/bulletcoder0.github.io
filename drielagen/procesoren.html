Processor


De processor ( of ook wel de CPU genoemd ) is een apparaat dat als de hersenen van de computer dient
DE processor heeft de taak om alle informatie die binnenkomt te verwerken
In moderne computers zit vaak een 32 of 64 bit processor dat betekend dat de procesor 32 of 64 bits tegelijkertijd kan verwerken.

Kloksnelheid


De kloksnelheid geeft aan hoe vaak per seconde een computer een signaal kan lezen, schrijven en kan verwerken.
De kloksnelheid van processors wordt gemeten in herts
Een kloksnelheid van 1 herts staat gelijk tot 1 bewerking per seconde
Een moderne computer heeft vaak 3 Gigaherts dat is dus 3000000000 bewerkingen per seconde
